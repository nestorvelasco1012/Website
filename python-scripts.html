File directory to CSV

This script is specifically created to generate CSV files by extracting file names from a designated folder. It proves particularly beneficial for organizing unpaid invoices stored as PDF files. Upon running the script, it generates a CSV file that lists the names of the unpaid invoices. This CSV file can be utilized to automatically populate an Excel sheet I developed, which fills in the relevant data for each invoice based on its file name. When you open the script, you have the option to select a folder. This prompts a file explorer, allowing you to navigate to the folder containing the desired file names that you wish to convert into a CSV format.

After selecting the folder, the path is shown to verify you have selected the correct folder. After veryfing the path, you click the "Create CSV" button, which will create and save the CSV in the same folder you have specified.

As you can see, the CSV file contains the names of the files in the folder you selected. This CSV file can be used to automatically populate an Excel sheet I developed, which fills in the relevant data for each invoice based on its file name.
