CSV File Merge

Generate CSV files by extracting file names from a folder.

This is a program that creates a graphical user interface (GUI) application using the Tkinter library. The purpose of the application is to merge multiple CSV files into a single file. When the program runs, a window is displayed with a label prompting the user to select the folder containing the CSV files. The user can click a "Browse" button to open a file dialog and choose the desired folder. The selected folder path is then displayed in an entry widget.

After selecting the folder, the user can click a "Merge CSV Files" button to initiate the merging process. The script checks if the specified folder exists and proceeds to read the contents of each CSV file in the folder. The data from each file is stored in a list.

Throughout the process, an error label is available to display any potential error messages. The window is centered on the screen, and the Tkinter event loop is started to handle user interactions and maintain the functionality of the GUI.
