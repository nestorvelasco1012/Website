Cross Check Invoices

Script used to cross check file names inside a folder with a csv file.

This program creates a graphical user interface (GUI) application using the Tkinter library. The purpose of the application is to select a CSV file and a folder, and then delete files from the specified folder based on the entries in the CSV file. When the program runs, it creates a window with the title "Delete Files". The window is centered on the screen and has a fixed size of 700x130 pixels. Within the window, there are three main components: a label and an entry field for selecting the CSV file path, a label and an entry field for selecting the folder path, and a button to initiate the file deletion process.

The user can click the "Select CSV" button to open a file dialog and choose a CSV file. The selected file path is then displayed in the corresponding entry field. Similarly, the "Select Folder" button opens a file dialog for the user to select a folder, and the chosen folder path is displayed in the entry field. Once both the CSV file and folder are selected, the user can click the "Delete Files" button. The script reads the CSV file to obtain a list of file names. It then iterates through the files in the specified folder and deletes any files whose names match the entries in the CSV list.

During the deletion process, the script displays a message box with the number of files deleted. If the CSV file or folder is not selected, an error message is shown instead. The script continues running in a loop until the window is closed, allowing the user to perform multiple file deletion operations.
