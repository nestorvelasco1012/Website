Tortilla Counter

Ultrasonic Sensor Counter

The project is a sensor-based tortilla counter designed to automate the process of counting and packaging tortillas as they slide down a slide. The system consists of two Arduino boards, an ultrasonic sensor, an LCD screen, a 4-way switch, a stepper motor, and several buttons for control. The ultrasonic sensor is positioned underneath the slide at a specific distance. When a tortilla slides through the sensor's range, it detects the change in distance and triggers a counter. This allows the system to accurately count the number of tortillas that pass through.

The LCD screen displays the current tortilla count and the packet count. The 4-way switch is used to set the desired packet size, which can be 10, 14, 20, or an infinite number of tortillas. This selection determines how many tortillas need to be counted before a packet is considered complete. When the tortilla count reaches the specified packet size, for example, 10 tortillas, the counter resets to 1, indicating the start of a new packet. At the same time, the packet count increases by 1, indicating the completion of a packet of tortillas. This information is displayed on the LCD screen.

Additionally, every time the packet count is reset, a signal is sent into another Arduino, and a stepper motor is activated. The stepper motor rotates a platform where the next packet of tortillas will fall into. This mechanism ensures that the packaged tortillas are neatly separated and ready for further processing. The main control box contains buttons that allow the user to increase, decrease, or reset the tortilla count. These buttons provide manual control over the counting process, allowing adjustments to be made if needed.

In summary, the project automates the process of counting and packaging tortillas using a sensor, an LCD screen, a 4-way switch for packet size selection, and a stepper motor. It provides an efficient and accurate solution for counting tortillas and packaging them into predefined packets, streamlining the production process.
