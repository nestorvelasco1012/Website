File directory to CSV

Generate CSV files by extracting file names from a designated folder.

This is a Python program that creates a graphical user interface (GUI) application using the Tkinter library. The purpose of the application is to select a folder and generate a CSV file that includes the names of PDF files within that folder. When the script is executed, a window is created with the title "Directory to CSV". The window is sized to 660x100 pixels and centered on the screen. Within the window, there is a label that prompts the user to select a folder. The folder path is inputted through an entry field. Next to the entry field, there is a "Select Folder" button. The "Select Folder" button is associated with the select_folder() function. When clicked, it opens a file dialog where the user can choose a folder. The selected folder path is then inserted into the entry field.

Below the folder entry field, there is a "Create CSV" button. Clicking this button triggers the create_csv() function. This function retrieves the folder path from the entry field and checks if a folder is selected. If a folder is chosen, the function creates a CSV file named after the selected folder on the user's desktop. It then writes the filenames of PDF files within the selected folder into the CSV file.

During the process, the script uses the messagebox module to display messages to the user. For example, a "Done" message is shown when the CSV file is successfully created, or an "Error" message is displayed if a folder is not selected.
